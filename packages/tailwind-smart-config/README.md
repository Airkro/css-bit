## Installation

```bash
npm install @css-bit/tailwind-smart-config --save-dev
```

## Usage

```cjs
const { tailwindSmartConfig } = require('@css-bit/tailwind-antd-color');

module.exports = {
  plugins: [tailwindSmartConfig()]
};
```
